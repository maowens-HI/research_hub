Sample Consistency: Jackknife vs Non-Jackknife

This analysis verifies whether the samples are identical between the non-jackknife spending charts (Figure 2.1) and the jackknife spending charts (Figure 2.4) from the 12/12 meeting.

Key Findings

  • Q0 (never-treated county-years) is identical across both specifications
  • Total observations unchanged at 83,384
  • Never-treated share remains 68.84% in both cases
  • The two tables are built on the same set of county-years with identical treated vs. never-treated composition
  • Differences arise only within the treated group: counties are reassigned across predicted spending quartiles when moving from non-jackknife to jackknife

Baseline vs Predicted Spending Cross-Tabulation

This analysis examines how baseline spending quartiles map to predicted spending quartiles.

  • If all rows are missing for a state, that state is never treated and therefore does not receive a predicted spending quartile.
  • If some rows are populated and others are missing, the state does not have enough counties in the balanced sample to form four quartiles (e.g., Arizona has only one county).

Key Pattern

Quartiles 1 and 2 of baseline spending tend to go to higher quartiles of predicted spending, while quartiles 3 and 4 tend to go to lower quartiles of predicted spending. This pattern is consistent with mean reversion in spending predictions.
