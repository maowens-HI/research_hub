Meeting Overview

This document presents the latest event study results comparing baseline (no jackknife) and jackknife approaches across three specifications:

  • Specification A: Baseline spending only
  • Specification B: Baseline spending + Income quartiles
  • Specification C: Baseline spending + Income quartiles * Reform types

Each specification is shown with both the full-sample estimates (baseline) and leave-one-state-out jackknife predictions.
